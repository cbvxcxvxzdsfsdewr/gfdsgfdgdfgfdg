# Random Password Generator

**Автор:** [Ердыгин Сергей]

**Дата:** [2026-04-25]

## Описание

Приложение для генерации случайных паролей с настраиваемыми параметрами и сохранением истории.

### Возможности:
* Настройка длины пароля (8–64 символа)
* Выбор типов символов: цифры, буквы, спецсимволы
* Генерация случайного пароля
* Сохранение истории паролей в JSON
* Отображение истории в таблице

## Проверка работы

1. Запустите приложение: `python password_generator.py`.
2. Протестируйте:
* генерацию пароля с разными настройками;
* проверку минимальной длины (должно быть ≥ 8);
* сохранение истории;
* перезапуск приложения (история должна загрузиться).
